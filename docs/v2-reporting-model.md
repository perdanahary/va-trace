# V2 Reporting Model

## Production Metrics

- Orders in production
- Orders in quality control
- Orders ready for distribution
- Production completion rate
- Production lead time

## Distribution Metrics

- Allocated quantity
- Shipped quantity
- Received quantity
- Delivery progress percentage
- Distribution success rate
- Exception count

## Sales Point Metrics

- Total Sales Points
- Sales Points with allocated quantity
- Sales Points dispatched
- Sales Points fully received
- Pending Sales Points

## Vendor Metrics

- Production SLA
- Distribution SLA
- POD compliance
- Shipment batch closure rate

## Derived Order Reporting Fields

Each order exposes:

- Production Status
- Distribution Status
- Delivery Progress
- Legacy Status Label for backward-compatible UI filtering
