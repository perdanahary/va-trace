# Plan: Split Production Pipeline Card into Two Separate Cards

## Context
In `src/pages/vendor/VendorOrderDetail.tsx`, the Overview tab (lines 475-608) has a single Card titled "Production Pipeline" that contains:
1. The `ProductionPipelineStepper` component
2. A "Ready for Shipping" table (jobs with `completedQuantity > 0`)
3. An "In Progress" table (jobs with `completedQuantity === 0`, not COMPLETED/CANCELLED)

The user wants these separated into two distinct cards based on the table data.
